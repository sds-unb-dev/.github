# Segurança

Se você identificar uma vulnerabilidade de segurança, não abra uma issue pública. Envie detalhes por email para: security@exemplo.com (substituir pelo contato real) ou utilize o canal indicado nos contatos do projeto.

Inclua informações sobre: versão afetada, passos para reproduzir, impacto e evidências (logs, PoC).

Responderemos com instruções para divulgar e mitigar a vulnerabilidade de forma coordenada.
