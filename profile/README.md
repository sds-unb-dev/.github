# Sala de Situação — UNB

Perfil organizacional da Sala de Situação da Universidade de Brasília (UNB). Centraliza padrões, templates e recursos compartilhados para repositórios institucionais.

## Principais tipos de projetos

- Templates e arquivos de governança (CONTRIBUTING, CODE_OF_CONDUCT, SECURITY)
- Workflows e automações compartilhadas
- Ferramentas de observação, dashboards e análise de dados
- Bibliotecas e utilitários comuns

## Como contribuir

- Abra uma issue para discutir mudanças relevantes antes de implementar.
- Envie pull requests pequenas, com descrição objetiva e referência à issue quando aplicável.
- Inclua testes e documentação sucinta para alterações funcionais.
- Siga as diretrizes de revisão e os responsáveis indicados em `CODEOWNERS`.
